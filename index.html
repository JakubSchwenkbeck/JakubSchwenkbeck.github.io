Spotlight: Behavior Classification for Autonomous Vehicles

This project uses a Recurrent Neural Network (RNN) to classify object behavior and assess the risk it poses to autonomous vehicles. By leveraging MATLAB and Simulink, it creates a robust model that helps improve the safety of self-driving cars.

  • Key Focus: Object behavior classification, risk assessment, autonomous driving.
  • Technology: MATLAB, Simulink, RNNs (LSTM), Java Integration.
  • Outcome: Achieved 95%-99% accuracy in classifying risky vs safe behaviors.
